Professional Works para XOOPS

Un módulo ideal para la publicación de portafolios profesionales en XOOPS, con útiles características para sacar el mejor provecho a tu portafolio.

Ver en GitHub

¡Gracias por descargar Professional Works! Por favor lee cuidadosamente los detalles e instrucciones siguientes para una correcta instalación del módulo

Requerimientos

  • XOOPS 2.5.7 (Se recomienda siempre tener la versión mas reciente de XOOPS).
  • Common Utilities 2.2

Esta versión de Professional Works ha sido desarrollada las últimas versiones de XOOPS y Common Utilities, por eso recomiendo siempre estar al día con las actualizaciones.

Instalación

Importante: Antes de poder instalar Professional Works es necesario contar con Common Utilities, instalado y configurado correctamente.

La instalación de Professional Works, una vez Common Utilites ha sido instalado, es similar a la instalación de otros módulos. A continuación se detallan los pasos a seguir:

  1. Extrae los archivos del paquete comprimido de Professional Works a tu disco duro.
  2. Sube la carpeta works, recién extraída, al directorio modules de tu instalación XOOPS. (Esto puedes hacerlo a través de un cliente FTP como FileZilla).
  3. Ingresa al panel de control de XOOPS. En este punto deberías poder ver el tablero de Common Utilities.
  4. Ingresa al administrador de módulos de Common Utilities.
  5. Localiza el logotipo de Professional Works en la barra lateral e instalalo haciendo clic en la opción Instalar.
  6. Revisa que todo haya funcionado correctamente.
  7. ¡Listo! Professional Works está instalado y listo para usarse.

Obtener Professional Works

Puedes obtener la versión mas reciente de Professional Works directamente desde GitHub en la sección releases.

También puedes obtenerlo haciendo un fork del repositorio en GitHub. De este modo también podrás contribuir con sugerencias y código para mejorar el módulo.

Actualizaciones

Professional Works, como casi todos los módulos desarrollados para Common Utilities, están preparados para recibir actualizaciones a través del sistema de actualizaciones integrado de Common Utilities. Esto significa que, una vez instalado el módulo, no deberás preocuparte por buscar manualmente archivos actualizados, si no que el módulo podrá recibir actualizaciones con un solo click.

Ayuda y reporte de errores

Para reportar errores y problemas con el módulo, por favor utiliza la herramienta de GitHub que se proporciona para tal objetivo. Es la única forma en la que me apoyo para dar soporte..
